HTML

HTML is the standard markup language for web pages. With HTML, you can create your own website.

CSS

CSS stands for Cascading Style sheets. CSS describes how HTML elements are to be displayed on the screen, in paper, or media. It saves a lot of the work. It can control the layout of multiple web pages at once. External stylesheets are stored in CSS files.

Basic HTML

  1. !DOCTYPE - Describes the document type.
  2. html - Defines the HTML document.
  3. head - contains information for the document
  4. title - Defines the title for the document
  5. body - Defines the documents body
  6. h1 to h6 - Defines HTML headings
  7. p - Defines a paragraph
  8. br - Inserts a single line break
  9. !--...-- Defines a comment

Images

  1. img - Defines an image
  2. map - Defines a client-side image mape
  3. area - Defines an area inside an Image

  4. figure - Specifies self contained content
