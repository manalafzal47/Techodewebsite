Comments in Python


In this section, you will learn about python statements, and how to correctly identify when to use them.

Comments are important because they describe why a piece of code was written. They can be written side by side of a block of code explaining what that code does, or on top of the code block. A great part of comments is that the python interpreter ignores the comments when it is executed. There are two types of comments in python. One is the single line comments, and the second is the multiple line comment.

In python, a single line comment can be written using a (#) hash symbol, followed by the comment.
