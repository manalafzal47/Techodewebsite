Java

Java is a popular programming language. It is very easy to learn and simple to use. It is open source, secure, fast and powerful. It is an object oriented language which gives us a clear structure to programs and allows code to be reused.

It is used for:

  1. Mobile applications

  2. Desktop applications

  3. Web applications

  4. Web servers and application servers
