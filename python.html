Python

Python is a high level programming language with dynamic semantics. Python’s simple, easy-to-learn syntax makes it easy to read, and makes it a great language for beginners to learn.

Comments

Comments are important because they describe why a piece of code was written. A great part of comments is that the python interpreter ignores the comments when it is executed. In python, a single line comment can be written using a (#) hash symbol, followed by the comment.

Variables

To store values in python, you can store them in variables. Variables are a placeholder for texts and numbers. It must have a name associated with it because it will store a value in it, and it will help when you want to reference it again. The variable is always associated with an equal sign, followed by the value of the variable. Some things to keep in mind are that variable names cannot have spaces. To separate words in variables, you can use an underscore. Variable names cannot be the same as built-in-functions, keywords, or reserved words in python. Below are guidelines for naming variables correctly: Variable names should be concise and descriptive. Use underscores to separate multiple words in variable names.

Numbers

Integers are numbers like -1, 1, 2, 3. They have type int in Python. You can use arithmetic operators on integers like (+, -, *, /) Addition: + Subtraction: - Multiplication: * Division: / To modify the order of operations, you can use parentheses.

Floats

Any number with decimals is considered a floating point number. Most of the time, you can floats just like integers.

Strings

A string is a series of characters. In python, anything inside quotes is a string. You can use either single quotes, or double quotes.

Concatenating python strings

When you place string liberals next to each other, it will concatenate them automatically. Example, greeting = ‘Good’, ‘Morning’ print(greeting) Output: Good Morning

Concatenating 2 string variables

When concatenating 2 string variables, you can use the operator (+)

Example, first_name = ‘John’ last_name = ‘Smith’ full_name = first_name + last_name print(full_name) Output: John Smith

Accessing string elements

Since strings is a sequence of characters, you can access the elements at an index. The first character in the string always has an index of 0. Example, string = ‘This is a string’ print string[0] print string[1] Python also lets you use negative indexing. If you use a negative index, python will return the character starting from the end to the string.

Getting length of a string

To get the length of a string, you can use the len() of a function. For example, str = ‘This is a string’ str_len = len(str) print(str_len)

Slicing Strings

Slicing allows you to create a substring of a string in python. For example, str = ‘This is a string’ print(str[0:2]
